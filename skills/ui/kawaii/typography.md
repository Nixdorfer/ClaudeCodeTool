# Kawaii Typography

## Font Family

- heading: Quicksand, Nunito, Noto Sans SC, Microsoft YaHei, sans-serif
- body: Nunito, Noto Sans SC, Microsoft YaHei, sans-serif
- mono: JetBrains Mono, Fira Code, monospace
- requirement: rounded typeface mandatory for all visible text

## Font Size Scale

- hero: clamp(2rem, 5vw, 3.5rem)
- h1: clamp(1.75rem, 4vw, 2.5rem)
- h2: clamp(1.3rem, 3vw, 1.8rem)
- h3: clamp(1.1rem, 2.5vw, 1.4rem)
- h4: clamp(1rem, 2vw, 1.2rem)
- body-lg: 1.05rem
- body: 0.95rem
- body-sm: 0.85rem
- caption: 0.75rem
- badge: 0.7rem

## Font Weight

- thin: 300
- regular: 400
- medium: 500
- semibold: 600
- bold: 700
- heading-default: 700
- body-default: 400
- emphasis: 600
- button: 700
- nav-link: 600
- badge: 600

## Text Color Mapping

- hero-title: text-heading (from color scheme)
- section-title: accent-secondary or accent-deep
- body-text: text-primary
- secondary-text: text-secondary
- muted-text: text-muted
- link-text: text-link
- link-hover: accent-primary
- inverse-text: text-inverse

## Line Height

- heading: 1.3
- body: 1.7
- compact: 1.4
- loose: 2.0

## Letter Spacing

- heading: 0.02em
- body: 0.01em
- wide: 0.05em
- button: 0.03em
- caption: 0.04em

## Text Decoration

- link-underline: none by default
- link-hover-underline: none use background color change instead
- heading-decoration: floral pseudo-elements flanking title

## Border Radius for Text Containers

- inline-code: 8px
- text-badge: 50px
- text-chip: 20px
- text-tooltip: 12px
- text-tag: 14px

## Paragraph Spacing

- margin-bottom: 1.2em
- max-width: 65ch for readability
- text-align: left for body center for headings

## List Style

- bullet-type: cute custom marker (flower or heart symbol)
- bullet-color: accent-primary
- list-indent: 1.5em
- list-item-spacing: 0.6em

## Selection Style

- selection-bg: accent-primary-at-0.2
