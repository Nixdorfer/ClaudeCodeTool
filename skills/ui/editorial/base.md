# 基本要求

## 核心特征
- 报纸排版风格 多栏布局 首字下沉 衬线体标题 报头装饰 经典印刷美学
- 排版即设计 字体选择/字号层级/行距是一切基础
- 颜色极度克制 黑白灰为主 仅一抹强调色点缀
- 所有装饰性元素 (圆角/阴影/动画) 降到最低 保持印刷品静态质感

## 页面要求
- 页面底色为新闻纸白 配合微弱纸张纹理背景 模拟真实纸面效果
- 纸张纹理使用 CSS 噪点或极低透明度的 repeating-linear-gradient 实现
- 页面最大宽度 1200px 居中 左右留白模拟报纸版心
- 英文标题字体 Playfair Display (Google Fonts) 经典高对比衬线
- 英文副标题字体 Libre Baskerville 或 Cormorant Garamond (Google Fonts)
- 英文正文字体 Source Serif 4 或 Merriweather (Google Fonts)
- 英文标签/分类字体 Inter 或 DM Sans (Google Fonts) 无衬线小字
- 中文回退字体 Noto Serif SC SimSun serif
- 图标使用线条插画风格单色 SVG 不使用 emoji
- 图标默认颜色跟随正文文字色 交互态跟随强调色

## 按钮形状
- 主按钮使用实心深色底 + 浅色文字 无圆角或极小圆角 (0-2px)
- 次要按钮使用透明底 + 深色边框 1.5px solid
- 文字链按钮使用下划线文字 无边框无背景 悬停时下划线展开
- 按钮文字使用无衬线字体 全大写 字间距 0.08-0.15em
- 按钮 padding 水平 28px 垂直 12px
- 所有 checkbox 使用方形勾选框 符合印刷品气质
- 单选按钮使用圆形空心圈 选中时实心填充

## Modal 规范
- 遮罩层使用 rgba(0,0,0,0.4) 无 blur
- 内容区使用新闻纸白背景
- 内容区顶部有一条 4px 粗线装饰 颜色为油墨黑
- 无圆角 保持方正
- 阴影使用 0 20px 50px rgba(0,0,0,0.12) 模拟纸张悬浮
- 所有 modal 都有不随滚动条滚动的顶部 header 放置标题 底部 footer 放置按钮
- 被滚动的内容放在中间 content 区域
- 关闭按钮使用无衬线小字 悬停时变为强调色

## 卡片规范
- 背景透明 不使用背景色
- 上下使用 1px 细线分隔 首张卡片顶部使用 3px 粗线
- 布局使用 grid 左侧文字 右侧缩略图 150px
- 卡片标题使用衬线字体 字重 700
- 卡片摘要使用正文字色偏暗
- 悬停时标题变为强调色 无位移无阴影变化

## 滚动条
- 尽可能不显示滚动条 如果放不下则滚动条需要自行实现美化
- 宽度 6px
- 轨道颜色跟随页面底色
- 滑块使用中灰色 无圆角
- 无发光无阴影

## 分隔线
- 细线 1px solid 中灰色 用于小节分隔 上下间距 30px
- 粗线 3px solid 油墨黑 用于大区域分隔 上下间距 30px
- 双线 3px double 油墨黑 用于最重要的分隔 (报头/版块) 上下间距 30px
- 装饰线 居中短横线 width 60px height 1px 用于段落间装饰

## 导航栏
- 固定顶部 实色背景 (新闻纸白) 无 blur
- 报头区域居中 使用衬线字体 2.5rem 字重 900
- 报头下方日期行 使用无衬线小字 全大写 字间距 0.1em
- 导航链接区域底部 2px 粗线
- 链接文字默认深灰 悬停时变为强调色
- 链接使用无衬线字体 全大写 字间距 0.08em
