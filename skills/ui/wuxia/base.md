# 武侠水墨风 基础规范

## 核心特征

水墨山水意境 宣纸底纹质感 毛笔书法字体 卷轴展开交互 印章装饰点缀 留白写意构图

## 页面

- 宣纸纹理背景 通过多层 radial-gradient 模拟纤维质感
- 页面边缘卷轴装裱效果 顶部/底部可加卷轴轴头装饰
- 远山意境层 用渐变模拟水墨远山 置于页面底部或顶部
- 大量留白 内容区域不超过页面 60%
- 内容区最大宽度 900px 居中
- 四角可加中式角花装饰 用 border 片段模拟

## 字体

- 标题: "Ma Shan Zheng", "ZCOOL XiaoWei", cursive 毛笔书法风
- 正文: "Noto Serif SC", "SimSun", "STSong", serif 宋体/楷体风
- 英文/数字: "Noto Serif", serif
- 标题 letter-spacing: 0.12em-0.2em
- 正文 line-height: 1.8-2.0
- 禁止使用无衬线字体

## 图标

- 使用水墨笔触风格图标 线条有粗细变化
- 图标颜色跟随文字色或强调色
- 尺寸: 小 16px 中 24px 大 32px
- 优先使用 SVG 可通过 stroke-width 变化模拟毛笔效果
- 关闭图标用中文字符 "关" 或 "x" 配合书法字体

## 按钮

- 主按钮: 朱砂底色 宣纸色文字 左右两侧加卷轴轴头伪元素
- 次按钮: 透明底 墨色边框 墨色文字
- 幽灵按钮: 无边框 文字带下划线装饰
- 所有按钮使用书法字体 letter-spacing: 0.1em-0.15em
- padding: 10px 32px
- 无圆角或极小圆角 (0-2px) 保持方正古朴
- hover 时主按钮加墨色阴影 次按钮边框/文字变强调色
- active 时 transform: translateY(1px)
- 禁止使用大圆角和渐变背景

## Modal 弹窗

- 卷轴展开动画进入 从中间向上下展开
- 背景叠加层: 半透明墨色 带轻微 blur
- 内容面板: 宣纸色背景 单线边框
- 内边距: 40px-48px
- 内部加一层内边框 (offset 8px) 用淡强调色
- 四角中式角花装饰
- 关闭按钮置于右上角 使用书法字体
- max-width: 520px

## 滚动条

- 宽度: 6px
- 轨道: 宣纸色
- 滑块: 淡墨色 border-radius: 3px
- 滑块 hover: 变为强调色
- Firefox 使用 scrollbar-width: thin 和 scrollbar-color

## 分隔线

- 使用 flex 布局 中间放古风符号
- 两侧渐变线 从透明到边框色再到透明
- 最大宽度 200px 每侧
- margin: 40px-50px 上下

## 卡片

- 宣纸色半透明背景 rgba
- 单线边框 边框色偏暖
- padding: 28px-32px
- 无圆角
- hover 时轻微位移 + 淡阴影
- 四角用伪元素加角花装饰 (border 片段 20px)

## 导航栏

- 固定顶部 宣纸色半透明背景 带 backdrop-filter blur
- 底部单线边框
- 品牌名使用毛笔书法字体
- 导航链接使用书法字体 hover 变强调色
- 链接下方可加圆点装饰 hover 时 scale 出现

## 表单输入

- 无边框设计 仅底部单线
- 背景半透明白
- focus 时底线变强调色 加粗到 2px
- 使用宋体/楷体字体
- placeholder 用淡墨色

## 设计禁忌

- 禁止大圆角 (超过 4px)
- 禁止毛玻璃大面积使用
- 禁止霓虹/荧光色
- 禁止无衬线字体作为主字体
- 禁止大面积纯色填充 保持留白
- 朱砂红仅用于点缀 不做大面积背景
