# 基本要求

## 页面要求
- 深空黑背景 全页面星空粒子效果 微弱星云径向渐变叠加
- HUD风格界面 所有面板带有扫描线纹理和角标标签
- 全息蓝数据面板 引擎橙作为主强调色
- 英文字体 Exo 2 (标题) + Inter (正文) + JetBrains Mono (数据/HUD)
- 中文字体 Noto Sans SC
- 图标使用空心线框单色SVG 风格统一为航天/HUD仪表盘图标 禁止emoji
- 页面固定层叠顺序: 星空粒子层(z:0) > 星云渐变层(z:1) > 内容层(z:10) > 导航栏(z:100) > 模态框(z:1000)

## 按钮
- 按钮使用直角切角形状 (clip-path 切掉左上右下两个角 各6px)
- 主按钮: 引擎橙渐变背景 白色文字 切角形状 发光阴影
- 次要按钮: 透明背景 全息蓝边框 全息蓝文字 切角形状
- 幽灵按钮: 透明背景 淡灰边框 用于低优先级操作
- 所有按钮文字使用 Exo 2 字体 大写字母 letter-spacing 0.08em
- 所有checkbox等是否选项都改用滑块替代 滑块使用引擎橙填充
- 二选一且必选的选项使用左右滑块替代

## HUD角标
- 所有面板/卡片/模态框左上角带有HUD标签
- 标签使用 JetBrains Mono 0.65rem 全息蓝色 大写 letter-spacing 0.15em
- 标签背景色与父容器底色一致 两侧各10px padding
- 标签带有与父容器一致的边框 圆角3px
- 标签定位: absolute top -10px left 20px

## Modal规范
- 遮罩层: 深空黑85%透明度 + blur(8px)
- 面板: 深空渐变背景 全息蓝淡边框 圆角12px
- 面板带HUD角标 (内容如 COMM LINK / DATA PANEL / SYSTEM 等)
- 固定顶部header放置标题 (Exo 2 字体 亮银色)
- 固定底部footer放置按钮
- 中间content区域可滚动
- 关闭按钮: 右上角30px圆形 引擎橙淡色背景+边框

## 滚动条
- 宽度6px 圆角3px
- 轨道: 深空黑色
- 滑块: 全息蓝暗色 悬停时加亮
- 尽可能不显示滚动条 需要时才显示

## 导航栏
- 固定顶部 深空黑85%透明度 + blur(15px)
- 底部1px全息蓝淡色边框
- 品牌名使用 Exo 2 800字重 亮银色
- 导航链接使用冷灰色 悬停变引擎橙

## 分隔线
- 使用全息蓝渐变: transparent > rgba(74,168,216,0.3) > transparent
- 高度1px 上下50px间距

## 卡片
- 深空蓝半透明背景 + blur(10px)
- 全息蓝淡色边框 圆角10px
- 悬停: 边框加亮 向上偏移3px 发光阴影
- 内边距28px
