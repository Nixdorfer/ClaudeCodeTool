# 基本要求

## 核心特征
- 赛博朋克风格 深蓝黑背景 霓虹色强调 切角设计 故障艺术
- 高科技低生活 (High Tech Low Life) 美学
- 暗中有光的克制感 霓虹色只用于强调元素 避免大面积亮色
- 布局倾向于不对称和错位感 但不影响可读性

## 页面要求
- 页面底色为极深蓝黑 配合微弱的网格线背景 模拟电路板/矩阵效果
- 背景网格使用 CSS 渐变实现 线条为主强调色 极低透明度 (0.03)
- 背景网格尺寸 50px x 50px
- 英文标题字体 Orbitron (Google Fonts) 几何感强的未来字体
- 英文正文字体 Rajdhani 或 Share Tech Mono (Google Fonts)
- 中文回退字体 Noto Sans SC Microsoft YaHei sans-serif
- 等宽字体 Share Tech Mono 用于数据类/代码类内容显示
- 图标使用空心线框单色 SVG 图标 不使用 emoji
- 图标默认颜色跟随文字颜色 交互态跟随强调色

## 按钮形状
- 所有按钮使用切角 (clip-path) 造型 不使用圆角
- 切角公式 clip-path: polygon(10px 0, 100% 0, calc(100% - 10px) 100%, 0 100%)
- 大按钮切角 10px 小按钮切角 6px
- 按钮默认透明背景 + 霓虹色边框 2px
- 悬停时内部填充半透明强调色 + 外发光
- 文字使用 Orbitron 字体 全大写 字间距 0.15em
- 所有 checkbox 等是否选项改用滑块替代
- 二选一且必选的选项使用左右滑块替代

## Modal 规范
- 遮罩层使用 rgba(0,0,0,0.85) 配合 backdrop-filter blur(5px)
- 内容区使用深蓝紫渐变背景 linear-gradient(135deg, #0d0d2b, #1a0a2e)
- 内容区边框为主强调色 1px solid
- 内容区使用六角切角 clip-path: polygon(0 0, calc(100% - 20px) 0, 100% 20px, 100% 100%, 20px 100%, 0 calc(100% - 20px))
- 所有 modal 都有不随滚动条滚动的顶部 header 放置标题 底部 footer 放置按钮
- 被滚动的内容放在中间 content 区域
- 关闭按钮使用副强调色 悬停时添加 text-shadow 发光

## 卡片规范
- 背景使用深蓝紫渐变 半透明
- 边框为主强调色 低透明度 (0.2)
- 使用六角切角 切角 15px
- 顶部有一条渐变发光线 从透明到主强调色再到透明
- 悬停时边框变亮 添加外发光 微微上移 3px

## 滚动条
- 尽可能不显示滚动条 如果放不下则滚动条需要自行实现美化
- 宽度 6px
- 轨道颜色跟随页面底色
- 滑块使用主强调色 配合同色外发光
- 下拉栏要做美化 要有卷动的动画

## 分隔线
- 使用双色渐变 从透明到主强调色到副强调色再到透明
- 高度 1px 上下间距 40px

## 导航栏
- 固定顶部 半透明深色背景 配合 backdrop-filter blur(10px)
- 底部边框为主强调色低透明度 (0.15)
- 链接文字默认冷灰白 悬停时变为主强调色 配合 text-shadow 发光
- 链接底部有从左到右展开的发光下划线动画
