# 用户界面书写规范

根据项目需求选择合适的风格 阅读对应目录下的子文件获取完整规范
每个风格目录包含: base.md(基本要求) colors.md(配色体系) effects.md(特效系统) typography.md(排版规范) animation.md(动画体系)
