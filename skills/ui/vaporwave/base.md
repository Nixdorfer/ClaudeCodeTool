# Vaporwave Base

## 核心美学

蒸汽波 (Vaporwave) 是一种根植于 90 年代互联网怀旧与消费主义废墟的视觉风格
核心意象: 粉紫黄昏天空 无限透视网格地面 日落圆盘 Win95 弹窗 希腊雕塑 棕榈树剪影 日文片假名装饰
整体氛围: 梦幻怀旧 而非黑暗未来 色彩饱和且梦幻

## 页面结构

```
┌─────────────────────────────────────┐
│  navbar (毛玻璃 底部紫光边线)        │

│    (深紫→暮光紫→深紫蓝→粉)          │
│                                     │
│         ☀ 日落圆盘 (横纹切割)       │
│                                     │
│  ┌───────────────────────────────┐  │
│  │    内容区 (毛玻璃卡片)         │  │
│  │    Win95 风格窗口边框          │  │
│  └───────────────────────────────┘  │
│                                     │
│  ═══════ 透视网格地面 ═══════       │

```

## 背景层级 (z-index 规划)

| 层级 | z-index | 内容 |
|------|---------|------|
| 天空渐变 | 底层 body background | 粉紫渐变 fixed |
| 日落圆盘 | 0 | fixed 半透明太阳 横纹切割 |
| 透视网格 | 0 | fixed 底部 45vh 透视变换 |
| 棕榈树剪影 | 1 | fixed 可选装饰 |
| 内容区 | 10 | relative 主体内容 |
| 导航栏 | 100 | fixed 顶部 |
| 弹窗遮罩 | 1000 | fixed 全屏 |
| CRT 扫描线 | 9999 | fixed 全屏 pointer-events: none |

## 圆角策略

所有组件统一 0px 圆角 (方角) 以贴合 Win95 复古风格
唯一例外: 日落圆盘使用 50% 圆角

## 间距系统

基础单位: 8px
常用间距: 8 / 16 / 24 / 32 / 48 / 64

| 场景 | 间距 |
|------|------|
| 组件内部 padding | 12-16px |
| 卡片 padding | 24-32px |
| 区块间距 | 48-64px |
| 导航栏 padding | 14px 32px |

## Win95 窗口组件规范

```
┌──────────────────────────────────┐
│ ▪ window.exe          [_][□][×] │  ← 标题栏 渐变紫→青
├──────────────────────────────────┤
│                                  │
│         窗口内容区                │
│     背景: 深紫半透明              │
│     边框: 2px solid 紫           │
│                                  │
└──────────────────────────────────┘
```

标题栏高度: 28-32px
标题栏背景: linear-gradient(90deg, #b967ff, #01cdfe)
关闭按钮: 18x18px 深紫底 白色边框
窗口边框: 2px solid 紫色强调色

## 装饰元素规范

全角字母间距: letter-spacing: 0.5em + text-transform: uppercase
日文片假名: 作为背景装饰文字 低透明度 (0.2-0.3)
希腊雕塑: 以 SVG 剪影或 emoji 形式出现 低透明度
棕榈树: fixed 定位于页面两侧 作为剪影 opacity: 0.15-0.25

## 响应式断点

| 断点 | 宽度 | 调整 |
|------|------|------|
| mobile | < 640px | 隐藏棕榈树 缩小日落圆盘 网格密度降低 |
| tablet | 640-1024px | 缩小日落圆盘 减少装饰 |
| desktop | > 1024px | 完整视觉效果 |
