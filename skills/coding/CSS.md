# CSS书写规范

- 使用kebab-case
- 每个选择器之间都有一个空行
- 每个分号后都有一个换行 且这一行有一个缩进
- 选择器与{同行
- }独占一行

# 示例CSS代码

le-pn {
	background: #242424;
	font-family: system-ui, sans-serif;
}
