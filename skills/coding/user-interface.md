# 用户界面书写规范

## 基本要求

- 页面禁止使用渐变色
- 按钮图标不要使用emoji 可以使用空心的单色线框风格SVG图标
- 英文字体优先选择Cascade Code
- 中文字体优先选择微软雅黑
- 所有checkbox都改用滑块替代

# 配色方案

所有的配色方案都有以下几个部分组成
- 背景色 大面积铺在背景上 作为底色
- modal色 用来当作展示在背景上的modal/panel等大型结构的背景色
- 按钮色 用来展示当作按钮的背景色
- 文字色 用来显示文字的颜色

## 亮色 白底蓝白配色

- 背景色 #f5f7fa
- modal色 #ffffff
- 按钮色 #4a69bd
- 文字色 #1a1a2e
